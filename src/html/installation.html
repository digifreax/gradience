Overview

Gradience UI is a latest Bootstrap flat Skin with more cool elemens that bootstrap make it more beatiful and easy to edit and use. :)

Download

First you go to purchase & download Gradience from CodeCanyon.

Integration

Gradience UI integration replacing or adding the bootstrap.css inside the head tag.

                <html>
                  <head>
                    <!-- Gradience stylesheet from package //-->
                    <link href="css/bootstrap.css" rel="stylesheet">
                  </head>
                  <body>
                    <div class="Container">
                      <!-- Your Content Here -->
                    </div>
                  </body>
                </html>
